Learning Scala and Playframework

I am learning the Play! framework for Java/Scala which is inspired by Ruby on Rails. I like Scala, but it's not an easy language to learn. Simple operations are cloaked behind open-ended syntax.

For example in the code below from the API Reference,

import play.api.mvc.Action

val echo = Action { request =>
  Ok("Got request [" + request + "]")
}



It took me a while to decipher what's really going on.


    

  1. scala methods can omit parenthesis
    2. 

  2. the apply() method of object x is called by x()
    3. 

  3. anonymous functions can be created by the syntax { x => x+1 }
    4. 



Adding these three together we see what's really going on,


val anon = { ... anonymous function ... }
val echo = Action.apply( anon )
