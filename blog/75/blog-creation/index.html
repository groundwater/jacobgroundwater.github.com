Blog Creation

I created this website with the following tools or services

  1. Ubuntu
  2. Rackspace Cloud
  3. WordPress
  4. lighttpd

Since the Rackspace instance only has 250MB memory, lighttpd performs much better than apache. The blog supports several ways to post comments. You can sign in using Facebook or Twitter, provided by

  1. Simple Facebook Connect
  2. Simple Twitter Connect

Costs

Other than time, my time, the only costs associated are the Rackspace hosting and GoDaddy domain registration. I prefer having my own virtualized image, in this case hosted in the cloud. It affords me root level access to the system to make any chances I deem necessary. The downside is an increased cost and I am ultimately responsible for the well being of the website.

  1. Rackspace Cloud Server - $15 / month
  2. GoDaddy Domain Name - $15 / year
Bandwidth costs are minimal, I'm hosting my blog photo stream from flickr so the bandwidth comes from their servers, not mine.

Collaboration

If anyone is interested in having the same blog format hosted by me, we can share the cost and perhaps even upgrade the server instance. It would have to be done my way, i.e. you get the same setup. Of course you'd have your own domain name and blog theme.
